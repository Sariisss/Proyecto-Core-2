# Decisiones del Corazón
https://sariisss.github.io/Proyecto-Core-2/

Juego web educativo (18–25 años) tipo “elige tu historia” para concientizar sobre el amor como **proceso** (no solo emoción), la **libertad dentro del amor** y la construcción de **relaciones sólidas** (confianza, comunicación y respeto), promoviendo virtudes como:

- Alegría
- Templanza
- Amabilidad

## Estructura

- `index.html`: interfaz del juego
- `style.css`: diseño minimalista y amigable con la paleta solicitada
- `script.js`: lógica del juego e indicadores

## Cómo ejecutar

Abre `index.html` en el navegador.

## Indicadores

- ❤️ Amor
- 🧠 Madurez
- 🤝 Confianza
- 😊 Bienestar emocional

Virtudes:

- Alegría
- Templanza
- Amabilidad

## Finales

El juego calcula un final según tus indicadores:

- Relación tóxica
- Relación superficial
- Relación sólida y consciente
